# UI/UX 개선 백로그

> 출처: 화면별 UI/UX ralph loop(휴리스틱 누적 비평 + 종합), 7화면 × 8개 UX 차원.
> 총 **55개** 발견. 인증 화면은 r1 스트림 타임아웃으로 미평가(0건) → 추후 재실행 권장.
> 상태 표기: ✅ 적용됨(이번 글래스 리디자인에 반영) · ⬜ 미적용(백로그).

## 화면별 발견 수
| 화면 | 건수 | 헤드라인 |
|------|------|----------|
| 진단 플로우 | 12 | 자동진행 피드백, 뒤로가기 리셋 오해, 저장 카드 배치, sticky 진행바 |
| 히스토리 | 13 | 위기 배너 가시성(안전), 타임스탬프·아이템 위계, 추천수단 패턴, sticky 탈출 경로 |
| 레이아웃+탭바 | 9 | 포커스 링·safe area·터치 타깃·활성 인디케이터 (P1 다수) |
| 랜딩 | 6 | 가치제안 톤(공감+기술)·메시지 중복, 게스트 경로 명확화 |
| 프로필 | 6 | 계정 관리 그룹핑, 로그아웃 로딩/에러 피드백, 탭바 밀착 여백 |
| 인증 | 0 | (미평가 — 재실행 필요) |
| 결과 리포트 | 0 | (별도 비평 0 — 진단 플로우에 통합 평가) |

## P1 (우선)
- ✅ **TabBar 키보드 포커스 표시 부재 + 터치 타깃/라벨 미달** (접근성) — `focus-visible:ring-2 ring-primary` 추가, 라벨 11→11.5px, 터치 타깃 ≥48px 확보.
- ✅ **Safe Area 미지원 — 탭바가 홈 인디케이터와 겹침** (모바일) — 탭바 `pb-[env(safe-area-inset-bottom)]`, 레이아웃 하단 여백 `max(7rem, 6rem+safe-area)`.
- ⬜ **게스트 저장 유도 카드가 결과(Report)보다 먼저 노출** (정서) — 저장 유도 카드를 Report 하단으로 이동, 배경 중성화(primarySoft→surface+line), 카피 완화('히스토리에 저장할까요? (필수 아님)') + '나중에' 버튼. _(현재는 Report 위 노출 — founder 확인 후 배치 변경 권장)_
- ✅(부분) **위기/법적 고지 가시성·배치 (안전 크리티컬)** — 위기 배너 `border-2 border-primary` + 알림 아이콘으로 가시성 강화 **완료**. ⬜ 법적 고지를 상황 선택 화면에서 제거하고 결과 화면으로 일원화 + 상황 선택엔 격려 카피('솔직하게 골라도 괜찮아요')는 **미적용**(현재 미성년엔 격려, 성인엔 고지 노출 중).

## P2
- ⬜ **설문 옵션 자동 진행 피드백 부재 + 재선택 방해** — ✅ 자동진행 200→320ms, 진행바 `duration-500 ease-out`, 선택 즉시 primarySoft 강조 **완료**. ⬜ 명시적 체크마크/bounce·'다음' 버튼은 미적용.
- ✅ **활성 탭 인디케이터 약함** — 활성 탭 `bg-primarySoft` 하이라이트 추가, strokeWidth 유지(떨림 없음).
- ⬜ **저장 상태 메시지가 긴 Report 스크롤에 묻힘** — sticky 배너/toast로 전환, '다시 저장' btn-ghost 승격.
- ⬜ **게스트 보호 경로 클릭 시 무반응(리다이렉트만)** — 클릭 시 '로그인이 필요해요' toast 또는 탭에 게스트 상태 시각화.
- ✅ **히스토리 빈 상태 과한 공백 + 약한 CTA** — primarySoft 아이콘 + 헤드라인/서브텍스트 + `.btn-primary` CTA 카드로 재구성, 패딩 축소.
- ⬜ **첫 문항 뒤로가기가 전체 진단 리셋(라벨 오해)** — '← 단계 변경' 라벨 또는 '뒤로(진단 초기화됨)' 마이크로카피, 확인 다이얼로그.
- ✅ **자유서술 강제성·버튼 라벨 모호** — '(선택) 자세히 적을수록 더 정확해져요' 안내 추가, 버튼 '진단하기'로 통일('건너뛰고~' 제거).
- ⬜ **선택지·나이대 버튼 터치 타깃·활성 상태 시각화 부족** — 44×44px 보장, 뒤로가기에 현재 선택값 표시.
- ⬜ **설문/Report 긴 스크롤 + 진행·탈출 경로 고정 부재** — 진행바·브레드크럼 sticky, Report에 sticky '← 히스토리'.
- ✅(부분) **히스토리 타임스탬프·아이템 위계 약함 + 탭 피드백 부재** — 타임스탬프 12px+캘린더 아이콘 승격, 아이템 `active:scale-[0.98]`+우측 화살표 **완료**. ⬜ 스테이지 배지화는 미적용.
- ⬜ **랜딩 가치제안 톤 차가움·메시지 중복·게스트 경로 모호** — ✅ '냉정한 거울'→'솔직하되 따뜻한' 순화 **완료**. ⬜ 미니헤더 통합·게스트 기대치 배너는 미적용.
- ⬜(부분) **Report 섹션 위계·추천 수단 패턴 모호** — ✅ '추천 수단' 마이크로 라벨(배지) **완료**(예: '추천 수단 — 카카오톡 가벼운 안부 한 통'), ✅ 점수 원형 라벨(`scoreTitle`: 고백 적정도/관계 안정도/재회 시도 적정도) **완료**. ⬜ 섹션 그룹화(점수+위기 / 액션·계획 / 근거·분석)는 미적용.
- ⬜ **프로필 정보 카드와 로그아웃/에러 그룹핑·여백 부재** — '계정 관리' 섹션 카드로 묶고 버튼 아래 여백, 에러 alert 승격.
- ✅(부분) **로그아웃 버튼 로딩/disabled 피드백 부재** — `.btn:disabled { opacity-50 + cursor-not-allowed }` 전역 적용 **완료**. ⬜ 스피너 아이콘은 미적용.
- ⬜ **판단 근거 ▲/▼ 델타가 판정처럼 읽힘** — 중립 라벨 병기('+5 신호') 또는 중성 배지화.

## P3
- ⬜ **나이대 grid-cols-2 텍스트 절단 + 랜딩 리스트 과다 스크롤** — 버튼 패딩/폰트 조정 또는 1열 전환, 랜딩 리스트 축소.
- ⬜ **TabBar backdrop-blur GPU 부하** — 저사양 기기 대비 `bg-surface/95` 대체 옵션. _(글래스 아이덴티티와 트레이드오프 — 현재는 글래스 유지)_
