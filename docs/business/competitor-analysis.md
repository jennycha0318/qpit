# 경쟁 분석

> PRD §11의 확장 문서. 상태: 🚧 초안

## 경쟁 구도
| 구분 | 예시 | 한계 |
|------|------|------|
| 범용 LLM | ChatGPT, Claude, Gemini | 입력품질·아첨·연속성·안전성 부족 (PRD §11-2) |
| 연애 상담 서비스 | 유료 상담, 전화 코칭 | 비쌈, 즉시성·접근성 낮음 |
| 콘텐츠 | 연애 유튜브/책 | 일반론, 개인화 없음 |
| 점술 | 사주/타로 | 근거 없음 |

## 우리의 위치
**개인화 + 즉시성 + 데이터 근거**를 동시에 충족 — PRD §11, §11-2 참조.

> 현재 MVP 기준: 진단 엔진은 결정적 규칙 기반(LLM 호출 아님)으로 "데이터 근거"를 제공한다. LLM 기반 개인화와 결과 피드백·데이터 플라이휠은 계획/미구현 단계다.
