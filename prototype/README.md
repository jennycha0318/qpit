# Prototype (HTML)

1단계: 빌드 도구 없는 순수 HTML/CSS/JS 프로토타입.
**목적: 진단 플로우(설문 → 리포트)의 UX·플로우를 빠르게 검증.**

## 실행
`index.html`을 브라우저에서 바로 열면 됨. (서버 불필요)

## 범위
- 실제 AI 호출 대신 **mock 데이터**로 진단 리포트 렌더링
- 화면 전환·설문 분기·리포트 레이아웃 검증에 집중

## 구조
```
prototype/
├── index.html   # 진입점
├── css/         # 스타일
├── js/          # 플로우 로직 (mock)
└── assets/      # 이미지·아이콘
```

검증이 끝나면 이 플로우를 `web/` (Next.js)로 옮겨 실제 AI와 연결한다.
